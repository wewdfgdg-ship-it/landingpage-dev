## 1. 핑크 그라디언트 (Dr.Melaxin 스타일)

```
Create a single landing page hero section for a premium Korean skincare brand.

Layout: full-width centered, vertical stack
Background: soft pink gradient (light pink top → rose pink bottom), with subtle floating pink sphere decorations
Content from top to bottom:
1. Subtitle text: "시멘트리 골조 케어 원리에 핑크 스피큘을 더해" (16px, gray)
2. Main headline: "안면부 전반의 꺼짐까지 케어" (36px, black, font-weight 900)
3. Product line name: "시멘트리 사이아노 핑크 스피큘 세럼 & 크림" (15px)
4. Highlight text: "핑크스피큘과 리보닉 특허성분의 시너지 효과" (17px, magenta bold)
5. Three certification circles (110px, dashed border, glass effect): "주름개선 기능성 인증", "미백 기능성 인증", "리프팅 개선 임상 완료"
6. Product image area (serum + cream bottles, centered)
7. Bottom 3-column footer bar: "피부 칼슘 REBORNIC 특허 원료" | "핑크 스피큘 고함량 함유" | "저자극테스트 완료"

Style: luxury Korean cosmetic brand, soft and feminine, Tailwind CSS, Korean text
Min-height: 100vh, mobile responsive
Do NOT use any external images - use placeholder divs for product images
```

```
Create a single landing page hero section for a Korean food/product brand with bold dark design.

Layout: full-width centered, vertical stack

1. Product icon/emoji centered (44px, white): 🧴
```
