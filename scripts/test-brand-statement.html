Clicks Mode

더 많은 단축키, 더 빠른 작업을 경험하세요

💡 휴식 조명 켜기 C + R
🚗 테슬라 프리컨디셔닝 C + T
🎵 AI 노래 검색 실행 C + S
🤖 ChatGPT 대화 실행 C + G
📷 카메라 실행하기 C + P

Clicks Mode란?

Clicks Mode는 36개의 키보드 버튼을 iOS 단축어와 연동하여
더 많은 작업을 빠르고 효율적으로 수행할 수 있습니다.

각 버튼에 자주 사용하는 앱을 실행하거나 스마트홈을 제어하고,
복잡한 워크플로우를 한 번의 클릭으로 실행하는 등 다양한 기능을 설정할 수 있습니다.
Keyboard는 단순한 타이핑 도구를 넘선 무궁무진한 가능성과 창의력을 지닌 Gear입니다.
