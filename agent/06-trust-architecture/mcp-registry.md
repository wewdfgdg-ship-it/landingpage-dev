# MCP 레지스트리 — Trust Architecture Agent

## 이 에이전트의 MCP 우선순위

| 순위 | MCP 서버 | 용도 | 사용 빈도 |
|------|----------|------|----------|
| 1 | Sequential | 규칙 로직 디버깅 시 (거의 안 씀) | 낮음 |
| 2 | Context7 | 라이브러리 문서 참조 시 | 낮음 |
| 3 | Magic | UI 관련 작업 없음 (규칙 엔진) | 낮음 |
| 4 | Playwright | E2E 테스트 없음 (규칙 엔진) | 낮음 |

> 이 에이전트는 규칙 엔진(AI 없음)이므로 MCP 서버 사용 빈도가 매우 낮다.

## Context7 — 라이브러리 문서 + 패턴

### 용도
외부 라이브러리 공식 문서, 프레임워크 패턴, 코드 예시 조회

### 이 에이전트의 주요 Context7 조회 대상
- **Next.js 16**: Route Handlers, 서버 컴포넌트 패턴 (API 라우트 구현 시)
- **TypeScript**: 유틸리티 타입, 타입 가드 패턴

### 에러 복구
- 라이브러리 못 찾음 → WebSearch → 직접 구현

## Sequential — 복잡한 분석 + 다단계 추론

### 용도
복잡한 디버깅, 규칙 로직 분석

### 이 에이전트에서의 사용
- **거의 사용하지 않음** — 순수 규칙 엔진으로 로직이 단순
- 예외: 규칙 매칭 로직이 복잡해질 경우 분석용

---

## Magic — UI 컴포넌트 생성

### 이 에이전트에서의 사용
- **사용하지 않음** — 규칙 엔진, UI 생성 없음

---

## Playwright — E2E 테스트 + 브라우저 자동화

### 이 에이전트에서의 사용
- **사용하지 않음** — 규칙 엔진, 브라우저 테스트 불필요
