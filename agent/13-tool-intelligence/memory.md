## 자동 업데이트 규칙

| 이벤트 | 갱신 대상 |
|--------|----------|
| 탐색 완료마다 | CHECKPOINT + SCAN_HISTORY + 소스 품질 |
| 추천 완료마다 | RECOMMENDATION_LOG |
| 브로드캐스트마다 | BROADCAST_LOG |
| 평가 완료마다 | REJECTED_LOG (미성택 시) |
| 의사결정마다 | DECISIONS |
| 에러 해결마다 | LEARNINGS + MISTAKES |
| 매 5회 | 성장 지표 재계산 |
| 피드백 분석 시 | Success/Failure Patterns + Best Chains + Deprecated + Upgrade Candidates |
| 소스 Tier 변경 시 | 소스 변경 이력 |

### 입력 블록 → 갱신 섹션

| 입력 블록 | 갱신 섹션 |
|----------|----------|
| [REVIEW_RESULT] | CHECKPOINT, RECOMMENDATION_LOG |
| [LOOP_SYNC] | 에러 패턴, LEARNINGS |
| [ESCALATION_RECORD] | MISTAKES, 블로커 |
| [NEW_TOOL] broadcast | BROADCAST_LOG |
| skill-feedback 변경 | Success/Failure Patterns |
