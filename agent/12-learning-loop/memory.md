### 프로젝트 특이사항
- 이 엔진은 AI ×1 (진단) + 규칙 기반 혼합 엔진
- 파이프라인 마지막 엔진: ⑪ Deploy 이후 비동기 실행
- ① Product Intelligence로 피드백 루프 형성 (WinningPattern)
- Level 3 처방은 반드시 사용자 승인 게이트 필요 (requiresApproval: true)
- A/B 테스트: 동일 섹션 동시 테스트 금지, 통계적 유의성 p < 0.05
- 최소 데이터: pageViews ≥ 100, 샘플 크기 ≥ 500
- 진단 임계값: bounceRate > 70%, scrollDepth.p50 < 30%, CTA < 2%, dwellTime < 2s, 전환율 < 1%
- 한국어 시스템 프롬프트 + JSON prefill 필수
- AI 진단 실패 시 규칙 기반 진단으로 fallback (부분 결과 반환)
- Phase 4 분석 단계: 5-15초 내 완료 목표
