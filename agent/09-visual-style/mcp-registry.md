# MCP 레지스트리 — Visual Style Agent

## 이 에이전트의 MCP 우선순위

| 순위 | MCP 서버 | 용도 | 사용 빈도 |
|------|----------|------|----------|
| 1 | **Magic** | 디자인 토큰 참조, 컬러 팔레트 생성 참고 | 중간 |
| 2 | **Context7** | Tailwind CSS v4 문서, 색상/타이포 패턴 | 낮음 |
| 3 | Sequential | 복잡한 디자인 분석 시 | 낮음 |
| 4 | Playwright | 시각적 검증 시 | 낮음 |

## Magic — UI 컴포넌트 생성

### 이 에이전트의 주요 Magic 사용 패턴
- **디자인 토큰 참조**: 무드별 색상 조합, 타이포그래피 스케일
- **컬러 팔레트**: 업종별 적절한 색상 조합 참고

## Context7 — 라이브러리 문서

### 이 에이전트의 주요 Context7 조회 대상
- **Tailwind CSS v4**: 커스텀 테마 설정, 색상 유틸리티
- **Next.js 16**: CSS 변수 주입 패턴
